System Overview

  • Authentication: Supabase email/password login
  • Session Persistence: LocalStorage for user and role data
  • Admin Security: PIN protected (1234)
  • Three User Roles:
    • 👤 Customer - Browse & book services
    • 🔧 Technician - View & update jobs
    • 👨‍💼 Admin - Manage system
  • PWA Features: Service worker for offline support
